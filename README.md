# Functional Features
- **랜덤 넘버 생성**
    - 사용자가 맞춰야 할 랜덤 넘버 생성 
- **사용자 입출력**
    - 3자리 숫자를 공백 없이 입력 받기
    - 볼, 스트라이크 수 출력
    - 게임 종료 시 다시 시작 혹은 완전 종료 입력 받기    
- **입력 판별**
    - 볼, 스트라이크 수 계산


# Required Coding Convention
- Indent Depth <= 2
- 3항 연산자(Ternary Op.) 사용 금지
- 메소드의 역할은 한 가지로
- 인스턴스 변수의 수 <= 2
- 메소드 인자 수 <= 3
- 코드 한 줄에 점 하나(Law of Demeter)
